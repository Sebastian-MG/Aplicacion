De que se trata:

Este proyecto busca que mediante el ingreso de un archivo CSV se le apliquen metodos de machine learning.

¿Que busca?

Calcular medidas de estadística como: Medidas de tendencia central como media, mediana, moda, medidas de dispersión como varianza, covarianza, dispersión y correlación. Esto mediante programación funcional.
